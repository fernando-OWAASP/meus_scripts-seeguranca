# 🚀 Nmap Automation Script

Este é um script simples em **Python** que desenvolvi para facilitar meu dia a dia estudando **Infraestrutura e Segurança**.

### 🛠️ O que ele faz?
O script funciona como um menu interativo para o **Nmap**, permitindo escolher entre:
1. **Scan Simples:** Identificação rápida de portas abertas.
2. **Scan Agressivo:** Varredura completa de todas as portas (`-p-`) e detecção de versões de serviços (`-sV`).

### 💻 Requisitos
* Linux (Testado em Rocky Linux/Kali)
* Python 3
* Nmap instalado

### 📖 Como rodar
```bash
python3 scan.py
